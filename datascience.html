Early Detection of Lung Cancer Using Data Science

Description --

With the help of Data Science Techniques and Machine Learning Algorithm developed a web based data science model to predict the detection of lung cancer in early stage based on their symptoms and routines. To build a data science model we have to perform certain steps.

Steps --

  1. Load and read the dataset
  2. Data Preprocessing
  3. Data Visualization
  4. Feature Importance
  5. Feature Selection
  6. Model Selection
  7. Train and Test the different models
  8. Model Evaluation
  9. Confusion Matrix
  10. Predictive System
  11. Model Deployment
